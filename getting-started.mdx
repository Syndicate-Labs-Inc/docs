This guide gets you from a fresh install to a ready workspace.

## Before you begin

You need:

- A supported desktop build of Syndicate for your operating system.
- At least one AI provider account: Claude, OpenAI/Codex, or Gemini.
- A local project folder where agents can safely work.

## Install Syndicate

1. Download the latest Syndicate installer from [usesyndicate.org](https://usesyndicate.org).
2. Run the installer and open Syndicate.
3. If your operating system asks for permission to run the app, approve it.

## Complete first-run setup

When Syndicate opens for the first time, it shows provider status and workspace setup. If no provider is connected yet, open **Settings > Providers** and connect at least one provider.

## Create your first team

After a provider is connected:

1. Click **Create Team** or open **Solo Agents** and choose **New Solo Agent**.
2. Choose a team workflow or solo-agent setup.
3. Select the local folder for the project.
4. Add or review agents.
5. Start a conversation with an agent and give it a small task.
