Most AI tools give you another chat box. Syndicate gives you a team.

Start with one request. A Manager can break it into steps, route work to specialists, keep context attached to your project, and bring the result back where you can see what happened. Use a full team when the work needs coordination, or a solo specialist when you want fast 1:1 help.

Why people choose Syndicate:

- **Real coordination, not tab juggling:** agents can message each other, hand off work, and report back through a Manager.
- **Your workspace stays visible:** chat, tasks, inboxes, references, and agent status live in one desktop app.
- **You choose the models:** connect Claude, OpenAI Codex, or Gemini and pick the right provider per agent.
- **Built around local work:** select the project folder agents should use instead of copying context into yet another web chat.

This page is the quick route. Each step links to a focused guide when you need more detail.

## The 5-minute path

<Steps>
  <Step title="Request access and install the app">
    Syndicate is in private beta. Request access, then install the desktop app from your invite email.

    [Install and sign in](/tutorials/install-and-sign-in)

    [Install and sign in](/tutorials/install-and-sign-in)
  </Step>
  <Step title="Connect one AI provider">

    [Connect providers](/tutorials/connect-providers)

  </Step>

    [Chat and dispatch](/features/chat-and-dispatch)
  </Step>
</Steps>

## Good first tasks

- "Read this project and summarize how it is structured."
- "Find the auth flow and explain it in plain English."
- "Review this folder and suggest the safest first improvement."
- "Draft a README section for running this project locally."

<Tip>
Start with a small, concrete task. Once the agent understands your project, give it larger work.
</Tip>
