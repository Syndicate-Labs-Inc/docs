This guide gets you from a fresh install to a ready workspace.

## Before you begin

You need:

- A supported desktop build of Syndicate for your operating system.
- At least one AI provider account: Claude, OpenAI/Codex, or Gemini.
- A local project folder where agents can safely work.

## Install Syndicate

1. Download the latest Syndicate installer from the website.
2. Run the installer and open Syndicate.
3. If your operating system asks for permission to run the app, approve it.

<Note>
  Screenshots for this section will live in `assets/screenshots/install`.
</Note>

## Complete first-run setup

When Syndicate opens for the first time, it checks which providers are available on your machine. If no provider is connected yet, open Settings and connect at least one provider.

## Create your first team

After a provider is connected:

1. Choose **Create Team** or **Let's Build a Team**.
2. Select the local folder for the project.
3. Add one or more agents.
4. Start a conversation with an agent and give it a small task.
