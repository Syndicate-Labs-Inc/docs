## Before you start

You need:

- An **approved beta invite email** with installer links.
- **macOS or Windows** to install the desktop app.
- An **internet connection** for sign-in and agent runs.
- An account with one supported provider — **Claude**, **OpenAI/Codex**, or **Google Gemini**.

Optional: install **Git** and configure your **Git credentials**. Neither is required for core setup, but they help with some Syndicate features and with working on projects outside the app.

<Note>
You don't need to install **Node** or **npm** yourself. Syndicate ships with a managed runtime and installs each provider's CLI on demand.
</Note>

## The 5-minute path

<Steps>
  <Step title="Request access and install the app">
    Syndicate is in private beta. Request access, then install the desktop app from your invite email.

    [Install and sign in](/guides/install-and-sign-in)
  </Step>
  <Step title="Create or sign in to your Syndicate account">
    Use the same email address that was approved for beta access.

    [Install and sign in](/guides/install-and-sign-in)
  </Step>
  <Step title="Connect one provider">
    Connect Claude, OpenAI/Codex, or Google Gemini. You only need one provider to start.

    [Connect providers](/guides/connect-providers)
  </Step>
  <Step title="Choose a workspace">
    Start with an Atlas-led team for coordinated work, or a solo agent for direct 1:1 help.

    [Quick start](/guides/create-your-first-team/quick-start)
  </Step>
  <Step title="Give a small first task">
    Ask Atlas or a solo agent to inspect your project, summarize the structure, or draft a small change.

    [Chat and dispatch](/features/chat-and-dispatch)
  </Step>
</Steps>

## Good first tasks

- "Read this project and summarize how it is structured."
- "Find the auth flow and explain it in plain English."
- "Review this folder and suggest the safest first improvement."
- "Draft a README section for running this project locally."

<Tip>
Start with a small, concrete task. Once the agent understands your project, give it larger work.
</Tip>
