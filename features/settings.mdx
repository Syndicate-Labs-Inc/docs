Settings is the control center for app-level configuration.

{/* Screenshot: Account settings showing signed-in user (email masked) and Sign out button. */}

## Account

Use Account settings to view your signed-in Syndicate account and sign out.

{/* Screenshot: General settings showing app data path, default provider, default model, and theme controls. */}

## General

Use General settings to manage:

- App data path.
- Default provider.
- Default model.
- Theme.

{/* Screenshot: Providers settings with the three cards (Claude, OpenAI/Codex, Gemini) in different states — one Connected, one Not signed in, one Not installed. */}

## Providers

Use Providers settings to:

- Connect providers.
- Disconnect providers.
- Refresh provider state.
- See installed and authenticated status.
- Choose defaults.

{/* Screenshot: MCP settings showing the catalog with both ready entries and docs-only entries, plus the manual-add control. */}

## MCP

Use MCP settings to:

- Review available MCP catalog entries.
- See setup requirements.
- Install or configure an MCP server.
- Add an MCP server manually.

{/* Screenshot: team settings inside an open team — rename, working directory, Manager control mode, and Hard pause controls visible. */}

## Team settings

Team settings live inside a team. Use them to rename the project, relink the working directory, choose Manager control mode, hard pause the team, or delete the team.
