---

The command palette is the fastest way to move around Syndicate.

## What it searches

The command palette can search:

- Teams.
- Agents.
- Conversation snippets.
- Common actions.

## Common actions

Actions include:

- Create New Team.
- Open Settings.
- Toggle Sidebar.
- View Chat.
- View Tasks.
- View Agent Config.
- Switch workspace layout.

## Tips

- Use the command palette when you have many teams or agents.
- Search by agent name when you want to jump directly into a conversation.
- Search for "settings" or "tasks" instead of navigating manually.
