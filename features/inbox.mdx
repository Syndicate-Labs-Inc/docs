Each agent can receive inbox messages from other agents. Inbox messages are how teams coordinate across specialists.

## What inbox messages are for

Agents use inbox messages to:

- Delegate work.
- Ask another specialist to review something.
- Report completion.
- Request clarification.
- Escalate a user decision to the Manager.

{/* Screenshot: sidebar showing unread badges on two agents and a total unread indicator on a team. */}

## Unread counts

Unread badges show which agents need attention. The app also tracks total unread inbox activity across teams.

{/* Screenshot: user-attention item in the inbox — a Manager message asking the user for a decision, with the reply control visible. */}

## User attention

When an agent needs your input, it can route the request through the Manager or surface it as a user-attention item. Treat these items as decision points — the team is waiting on you before it can continue cleanly.

## Clearing confusion

If messages pile up or the team seems stuck:

1. Open the inbox.
2. Read the latest Manager messages first.
3. Stop any agent doing the wrong work.
4. Give the Manager a clear next instruction.
