References are files or folders you attach so agents can use them as context during a run. Attaching the right references — at the right scope — is one of the biggest levers on agent quality.

## Pick the right scope

| Scope | Visibility | Use for |
| --- | --- | --- |
| **Global** | All workspaces, all agents | Company style guide, shared standards |
| **Team** | All agents in one team | Project README, architecture doc, schema |
| **Agent** | A single agent | Files only that specialist needs |

Start at the narrowest scope that works. Global references load on every run, so reserve them for material every agent genuinely needs.

## Attach a reference

{/* Video: add a reference at one scope and confirm it's attached. */}

<Steps>
  <Step title="Choose the scope">
    Decide whether the reference belongs at **global**, **team**, or **agent** scope.
  </Step>
  <Step title="Add a file or folder">
    From the relevant references control in Syndicate, attach a file (for example a README or schema) or a folder (for example a `docs/` directory).
  </Step>
  <Step title="Confirm it's attached">
    Check that the reference is listed for that scope before starting a run.
  </Step>
</Steps>

## Verify the reference reaches a run

{/* Screenshot: an agent run referencing the attached file. */}

After attaching, send the agent a small task that requires the reference — for example, "Summarize the architecture doc I just attached." If the agent answers from the document, the reference is wired up. If it asks for the file or guesses, recheck the scope.

## Keep context focused

More context is not always better. Large or off-topic references crowd the prompt and can push more useful information out.

- Attach the smallest set that answers the task.
- Prefer a specific file over a whole folder when one file is enough.
- Detach references the agent no longer needs.

## Context budgets to know

Syndicate inlines reference content with limits so prompts stay healthy. Very large files, deeply nested folders, or many references combined may be truncated when they reach the prompt.

If an agent seems to be missing details from a reference, split large material into focused files rather than attaching one giant document.
