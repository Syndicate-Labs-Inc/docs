## What Syndicate helps you do

- Build a team of AI specialists around a local project folder.
- Give one request to a Manager and let it coordinate specialists.
- Work 1:1 with a solo agent when you want speed and focus.
- Connect Claude, OpenAI Codex, and Gemini, then choose models per agent.
- Add context through references, skills, MCP tools, and project files.
- Keep work visible with chat, tasks, inboxes, and agent status.
