MCP lets agents access external tools and data sources.

## Open MCP settings

Go to **Settings > MCP** to review available MCP servers.

## Configure a server

Each server has its own setup requirements. Some need API keys, OAuth credentials, or environment variables.

## Enable MCP for an agent

After a server is configured, open an agent and enable the MCP server for that agent.

## Use a focused request

Ask the agent to use the connected source for a specific task:

```text
Use the connected docs source to find setup instructions and summarize the exact steps.
```

## Result

The agent can use the enabled MCP server during its run, while other agents remain limited to their own configured tools.

For setup guidance and safety tips, see [MCP](/features/mcp).
