## How to use these demos

Each demo describes the expected path, the important decision points, and the result you should see when the workflow is complete.

For detailed instructions, use the matching tutorial or feature reference linked from each page.

## Recommended first demo

Start with [Create your first team](/demo/first-team) if you want to see Syndicate's multi-agent workflow. Start with [Use a solo agent](/demo/solo-agent) if you want the fastest path to a useful first run.
