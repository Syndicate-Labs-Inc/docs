This demo shows the basic path from an empty workspace to a working team.

## Start from the workspace screen

Open Syndicate and choose the team creation path. You can start from a template, describe your project, or build from scratch.

Templates are best when you already recognize the workflow you need. The guided description path is best when you want Syndicate to suggest the team shape.

## Choose a project folder

Select the local folder where the team should work. This becomes the working directory for agents.

## Review the team

Before creating the team, review:

- Team name.
- Manager model.
- Specialist agents.
- Agent models.
- Working folder.

## Create the team

After the team is created, select an agent and send a small first task.

## Example first request

```text
Inspect this project and summarize what it does, how to run it locally, and what files I should understand first.
```
