Solo agents are the fastest way to start working with Syndicate.

## Choose solo agent

From onboarding or workspace creation, choose **Solo Agent**.

## Pick a specialization

Choose the role that matches the work:

- Frontend Developer for UI work.
- Backend Developer for APIs and server logic.
- QA Engineer for test plans and regression checks.
- Technical Writer for docs.
- Researcher for investigation and synthesis.

## Choose a model

Pick an available model from your connected providers.

## Start with a focused request

Good solo requests are specific:

```text
Review the settings UI and suggest the three highest-impact improvements.
```

## Result

The solo agent opens as a direct chat workspace tied to your selected project folder. Use it for focused work that does not need delegation.

For more context, see [Solo agents](/features/solo-agents).
