## Syndicate account

- **Sign-up** requires full name, email, password, and email confirmation. Confirmation links open the app via the `syndicateapp://` deep link and complete sign-in automatically.
- **Sessions** are stored encrypted on-device (OS keychain-backed storage, never plain localStorage) and restored on launch. Tokens refresh automatically; if refresh fails or the session is revoked, you are returned to the sign-in screen.
- **Sign-out** clears the encrypted session locally first, then revokes the token best-effort, so sign-out works even offline.
- **Devices**: each install has a device identity ("Syndicate Desktop (Windows)" and similar). Account tiers may limit concurrent devices; if exceeded you will be asked to sign out elsewhere first. Closing the app releases its device session.

## Launch policy

On startup Syndicate checks a launch policy for your account and version. It can require a minimum app version (with an update link), an active subscription, or specific tiers. If the check cannot be reached (offline), the app launches normally and re-checks later.

## Provider auth flows

### Claude (browser code flow)

1. **Connect** opens claude.ai in your browser with a secure one-time challenge.
2. You approve access and copy the displayed code.
3. Pasting the code into Syndicate exchanges it for tokens, which are written to the Claude CLI's own credentials file (`~/.claude/.credentials.json`).
4. On rate-limit errors (HTTP 429) the app automatically falls back to a terminal `claude` login and watches the credentials file for completion.

### OpenAI and Gemini (terminal flows)

1. **Connect** launches the provider CLI's interactive login (in-app terminal on Windows/Linux, Terminal.app on macOS).
2. You complete the provider's own sign-in (ChatGPT account for Codex, Google account for Gemini).
3. A file watcher polls the credentials location for up to 5 minutes; on success the provider card refreshes automatically.

### Failure modes

| Message | Meaning |
|---|---|
| "Invalid code — try again." | The pasted Claude code was wrong or expired (about 15 minutes); get a fresh one |
| "Auth terminal exited with code X before completion." | The CLI login was interrupted; close and click Connect again |
| "Claude OAuth endpoint is busy — switching to terminal login." | Automatic fallback in progress; finish in the terminal |
| "Install failed." | CLI install failed; the diagnostic modal lists each step, with **Copy diagnostic** for support |
