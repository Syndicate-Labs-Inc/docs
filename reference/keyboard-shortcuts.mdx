title: "Keyboard shortcuts"
description: "Every keyboard shortcut in Syndicate."
---

Press **?** anywhere to open this list in-app.

## Navigation

| Shortcut | Action |
|---|---|
| **Ctrl+K** | Open the command palette |
| **Ctrl+B** | Toggle the sidebar |
| **Ctrl+[** | Previous agent |
| **Ctrl+]** | Next agent |
| **Esc** | Close the current panel or modal |

## Chat

| Shortcut | Action |
|---|---|
| **Enter** | Send message |
| **Shift+Enter** | New line |
| **/** (in the input) | Open the slash-command menu |
| **Ctrl+V** | Paste an image or screenshot as an attachment |

## Command palette

| Shortcut | Action |
|---|---|
| **Up / Down** | Move selection |
| **Enter** | Select |
| **Esc** | Close |

## Forms and modals

| Shortcut | Action |
|---|---|
| **Ctrl+Enter** (Cmd+Enter on Mac) | Submit (feedback form, task assignment, team generation) |
| **?** | Show the keyboard shortcuts panel |

On macOS, use **Cmd** wherever **Ctrl** is listed.
