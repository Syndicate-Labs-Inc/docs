Starter templates create a fully configured team in one click. Every template includes Atlas as Manager (default model: Fable 5) plus the specialists below. Everything is editable on the Review screen and after creation.

## Application Developers

"Full-stack development — architecture, frontend, backend, testing, and security"

| Agent | Specialization | Default model | Focus |
|---|---|---|---|
| System Architect | Backend Developer | Fable 5 | Architecture, tech stack decisions, API contracts, codebase coherence |
| Backend Developer | Backend Developer | GPT-5.3 Codex | Services, APIs, database logic, auth, business rules |
| Frontend Developer | Frontend | GPT-5.3 Codex | Components, pages, state management, responsive design |
| QA Engineer | Test Engineer | Sonnet 4.6 | Unit, integration, and end-to-end tests |
| Security Engineer | Security | Sonnet 4.6 | Vulnerability review, dependency audits, hardening |

## Customizing a template

On the Review screen you can toggle agents off, rename them, change specializations and models, and add agents before creating. After creation, everything remains editable per agent. Templates are starting points, not contracts.
