---
title: "Settings"
description: "Settings areas users commonly need while setting up Syndicate."
---

Settings is where users connect providers, choose models, and adjust app-level configuration.

## Provider settings

Use provider settings to:

- Connect or reconnect Claude, Codex, and Gemini.
- Inspect provider readiness.
- Resolve authentication problems.
- Choose available models for agents.

## Team settings

Use team settings to:

- Rename teams.
- Update project folders.
- Adjust team-level instructions.
- Review agent configuration.

## Agent settings

Use agent settings to:

- Change provider or model.
- Edit role and behavior instructions.
- Add or remove skills.
- Tune how the agent reports progress.
