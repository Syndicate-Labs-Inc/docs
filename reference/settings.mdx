A consolidated map of every user-facing setting. The guided tour is in [Settings](/features/settings).

## App settings (Settings modal)

### Account

| Setting | What it does |
|---|---|
| Profile | Shows your name, email, and tier badge (free, beta, pro, gold) |
| Team limit / Solo limit | Your tier's caps on teams and solo workspaces |
| Sign out | Clears your session on this machine (local data is kept) |
| Manage subscription | Not active yet ("Billing and plan management arrives in Phase 2.") |

### General

| Setting | Default | What it does |
|---|---|---|
| Theme | Dark | Light or Dark, applied immediately |
| Confine agents to their working folder | Off | OS-level sandbox restricting non-manager agents to their team's folder (macOS/Linux; not applicable on Windows). See [Permissions and safety](/reference/permissions-and-safety#workspace-confinement-global) |
| Check for updates / Restart now | automatic | Manual update check; restart installs a downloaded update |

### Providers

| Setting | What it does |
|---|---|
| Default Model For New Agents | Provider + model used by new teams and solo agents; existing agents keep theirs |
| Provider Health cards | Per provider: status badge, CLI version, models, and Install CLI / Connect / Reconnect / Disconnect / Refresh actions |

### MCP

| Setting | What it does |
|---|---|
| Installed servers | Health, transport, Test, enable/disable, Remove per server |
| Add a server | Manual install: stdio command, HTTP URL, or SSE URL, plus encrypted secrets |
| Catalog | One-click presets; credential-requiring presets prompt first |

## Team settings (gear on a team)

| Setting | What it does |
|---|---|
| Project Name | Rename the team |
| Working Directory | Relink the team to a different folder (agents and history kept) |
| Interruption Level | Minimal / Balanced / Hands-on / Manual: how often Atlas stops to ask you |
| Team Hard Pause | Stop dispatches and block auto-dispatch until resumed |
| Delete Team | Permanent removal of the team and its agents (project files untouched) |

## Agent settings (configuration panel)

| Setting | What it does |
|---|---|
| Name / Specialization / Model | Identity and the model it runs on |
| Soul | The agent's system prompt (editable for user-made agents) |
| Context | Presets (Lean / Balanced / Rich / Custom), history lookback, char limits, token budget, section toggles, Prompt Analytics |
| Skills | Read-only list of the agent's slash-command skills |
| Tools (MCP) | Per-agent toggles for installed MCP servers |
| Delete Agent | Removes the agent and its configuration |

## Per-agent permission mode (shield selector)

| Mode | Behavior |
|---|---|
| Ask first | Prompt before every tool use |
| Auto-accept edits | File operations automatic; commands prompt |
| Full auto | No prompts |

## Other persistent preferences

| Preference | Where set |
|---|---|
| Show updates from agents | Checkbox in Atlas chat, remembered per team |
| Sidebar visibility | Ctrl+B, remembered |
